# Chronos

## Decentr Blockchain Explorer
* Blocks
* Transactions
* Validators

## Preparation to run
1. Install NodeJS.
1. Make a clone of Charon repository.
1. Install npm dependencies:
```bash
npm i
```

## Run
```bash
npm run start
```

## Build
```bash
npm run build
```

## Build (production)
```bash
npm run build:prod
```
